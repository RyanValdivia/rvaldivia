Mi primera página web

Esta tarea, consiste en desarrollar un sitio de páginas web personal, utilizando HTML y CSS, conteniendo información real de cada uno.

En este trabajo, tuve que explorar e investigar diferentes fundamentos de la programación web, desde la estructura esencial proporcionada por HTML, hasta el diseño estético y estilización mediante CSS.

Considero que aún me falta mucho por aprender y dominar en este enorme campo de la programación web, pero pondré todo de mí para seguir mejorando.
